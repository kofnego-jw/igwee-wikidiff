Welcome

This webapplication should be able to view revisions of a wikipedia entry and use the diff function to view the difference between revisions.

Usage

First you need to download a Wikipedia Export file which contains the revisions you want to review and upload it to the application. Or you can use the import (download) function of the web app

After uploading the file (it could take a long time), use the the view tab to view each revision, the diff tab to view differences and the search tab for fulltext search within the revision file.

Requirements:

  • Java 8 SE
  • This program as JAR
  • A web browser
